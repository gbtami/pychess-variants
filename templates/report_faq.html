Report FAQ

When should I submit a report?

Submit a report for cheating, harassment, hate speech, threats, spam, impersonation, or other serious abuse that impacts fair play or community safety.

What should I include to help moderators?

Include concrete evidence: game IDs, message snippets, and clear context. Reports with specific links and facts are reviewed faster and more accurately.

Why do inbox reports have different reason choices?

Inbox reports focus on communication abuse. Profile/game reports allow additional categories such as cheating-related behavior.

How many messages should I select in an inbox report?

Select only the messages that demonstrate the issue. Avoid selecting unrelated chat, so moderators can review the problem quickly.

What happens after I send a report?

Moderators review the report queue and may open an inquiry, apply moderation actions, or close the report if no violation is found.

Can I report the same user multiple times?

Yes, if there are separate incidents. For one incident, prefer one detailed report over many short duplicates.

Should I block the user too?

If the issue is direct harassment or spam, blocking can immediately reduce contact while moderation review is pending.

Will my report text be visible to other players?

No. Reports are for moderators/admins only and are handled in the moderation queue.
