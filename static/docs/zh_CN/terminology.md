# 术语介绍

本站的大部分地方，尤其是规则说明页中，用到了许多棋类术语。大部分术语是所有游戏通用的，但少数词语只适用于一部分游戏（例如Byo-yomi），也有一部分英文术语在不同的棋中表达不同的意思，例如"Stalemate"在国际象棋中译为“逼和”，而在中国象棋中则为“困毙”。
本文将展示常见的一些术语。关于某种棋类中的特有术语（例如将棋中的打入等），请查看相应的页面。

# “象棋类游戏”与“变体”

译者发现，"Chess Variants"这个表达，广泛在用于称呼各种象棋类游戏。然而，直译成“国际象棋变体”会出现误解。诚然，双狂象棋(Crazyhouse)、菲舍尔任意制象棋(Chess 960)等确实可以称之为“国际象棋的变体”，它们也确实是国际象棋的衍生游戏。然而，例如中国象棋、将棋、朝鲜象棋、泰国象棋等，也被称为"Chess Variants"，由此可见，简单的认为"Chess Variants"这个称呼是“国际象棋变体”并不合适。

在ChessVariant（最大的棋类游戏收集网站）上有一篇文章专门讲述这个问题：["What is a chess variant?"](https://www.chessvariants.com/what.html).文章中从国际象棋出发，提出了常见的棋类游戏的一些共同特征，并且提出"Chess Variant"。在PyChess里，"Chess Variant"的概念也使用了这篇文章的观点。简单来说，"Chess Variant"指一类回合制、拥有不同走法能力的作战单位（棋子）、以将杀对方特定棋子（例如王）为胜利条件的抽象策略游戏。现代的国际象棋通常认为源自印度的“恰图兰卡”(Chatturanga)，而虽然中国、日本、东南亚等地的棋类游戏与国际象棋大不相同，但是它们也与恰图兰卡有或多或少的联系。因此，译者认为，"Chess Variant"中的"Chess"并不单指国际象棋，而指的是更广泛的棋类游戏——中国的新棋圈中，把各种各样的棋分为擒王、灭绝、成形、占地、连接、堵截、到达等不同的类型，而在这里显然指擒王棋——所以此处中文翻译使用“象棋类游戏”的称呼。

中文里，“变体”(Variant)往往指直接由某种棋衍生而来的、与原本棋相似度较高的棋类游戏，而独立的设计棋盘、单位能力、规则等而成的棋类游戏（尽管其中可能会参考已有的棋类机制），往往不会被归类为变体。在中国有“新棋”的概念，然而这个概念有时候也比较模糊，它可以指新出现的变体，也可能指一个全新的棋。“新棋”像是一个时间概念而非游戏性上的概念，但同时“新棋”也包含了“新的设计”的意味。这和"Chess Variant"也有类似的模糊性。大部分时候，看"Chess Variant"究竟是指一种新的棋类游戏还是严格指国际象棋变体都可以从上下文中推断。如同“新棋”一样，"Chess Variant"也存在歧义的可能。部分人认为应当使用一个更通用的术语"chess-like variant"，但这种说法尚未作为标准术语。

# 棋子

**仙灵棋子(Fairy piece)** - 很多对仙灵棋不了解的人可能有疑问仙灵棋是个什么样的棋，但是**仙灵棋(Fairy Chess)** 并不是一个特定的棋盘游戏，而是一个棋类游戏中的概念。它最初是由T.R.Dawson在1918年提出，最初是出于一个“在棋盘谜题中引入更多自创棋子”的想法，从国际象棋出发，逐渐形成了一个庞大的棋子库，即为仙灵棋子。“仙灵棋子”即指在原版游戏中并未使用而在衍生玩法中新引入的棋子。广义的“仙灵棋子”也指在棋类游戏中使用的棋子的统称。

同一种能力的棋子，在不同的棋中可能具有不同的名称。例如中国象棋中的车，在将棋中叫做飞车；不同棋中可能出现相同的名字（一般为简便而采取的译名），如中国象棋和国际象棋的【象】指的是不同走法的棋子（后者在可能引起歧义的情况下，一般称为主教）。

## 棋子分类

仙灵棋子收录了海量的棋子。它们的分类方法多种多样。在英文文章中，经常会出现一种叫做BFN(Betza's Funny Notation)的缩写体系，用于描述棋子走法。如有兴趣，可搜索相关文章查看，在这里不再赘述。下面介绍一些常见的分类：

**长距离棋子/线性棋子(Riders)** - 指棋子可以向着某个方向连续移动直到遇到阻碍或到达棋盘边缘为止。如中国象棋的车、国际象棋的皇后。

**跳跃棋子(Leapers)** - 指一个棋子拥有固定的走法（区别于长距离棋子），且移动到目标位置时不受任何棋子的阻碍。一些棋类网游中使用了非常现代化的词语"Teleport"表示这一特性。国际象棋的马显然属于此类，但中国象棋的马严格来说不属于此类，因为蹩马腿的设定，它实际上不是“一步到位”——有的文章会以"lame leapers"称呼。

**炮系棋子(Hopper)** - 虽然Hop直译也是跳跃，但是Hopper指的是行动时必须跳过路径上一枚其他棋子的这一类棋子。（尽管现在也有必须跳过两枚棋子吃子的设计出现，使得“一枚”这个表述也有所改变。）一个典型的棋子便是炮。炮的吃法是标准的炮系走法。当然，朝鲜象棋的“包”则是更“纯粹”的炮系棋子：它走子吃子都必须隔子进行。

**复合棋子(Compound pieces)** - 如果一个棋子可以看成同时具备其他两个或多个棋子的走法，那么这个棋子就可以认为是复合棋子。例如国际象棋后=车+象，将棋中龙王=飞车+玉将。事实上，大部分仙灵棋子都可以认为是多种棋子的复合。

**走吃分离棋子(Divergent pieces)** - 如果一个棋子的走子方法和吃子方法不同，那它就属于走吃分离的棋子。典型例子是国际象棋的兵、中国象棋的炮。走吃分离大大扩充了仙灵棋子的棋子可能性，很多棋类网游中也以此设计出了大量的扩展棋子。

# 时间限制

如同大多数下棋平台一样，PyChess上的对局通常设定时间限制，也就是“快棋”模式。与快棋相对的则是“慢棋”，英文称为"Correspondence game"。慢棋一般通过发帖或邮件通信进行，每一步的时间比快棋长，有可能是几天。PyChess目前只有快棋，暂不支持慢棋。

如同棋钟一样，双方拥有各自的时间限制。下面是三种计时方式：

1.**补时制（菲舍尔计时制）** - 每次玩家结束他的行动，该玩家获得一定量的补时。例如：对局计时设定为"10+15"，意味着双方初始的时间限制为**10分钟**，而每次走棋之后补时**15秒**。大部分棋的设定使用此方式。

2.**读秒制(Byo-yomi，即秒読み，日语读秒)** - 在将棋和朝鲜象棋中经常使用的计时制。当玩家的时间（将棋称持子时间）用尽之后，接下来的每一步都必须在规定时间内完成行动。若超时，则判负。例如"10+30(b)"代表玩家初始的时间为10分钟，当10分钟用尽之后，每一步必须在30秒内完成。为了使玩家有足够的时间处理应急的复杂状况，也允许设定为多次超时之后才判负。例如"10+3x30(b)"即允许玩家超时，但超时第三次才判负。

3.**突然死亡** - 不会有额外加时，也没有读秒。一旦玩家耗尽时间立刻判负。读秒式的棋类可以将读秒时间设为0来使用此方式。

# 通用概念与术语

**将军(Check)** - 用棋子攻击对方的王，威胁下一回合将之吃掉。在擒王棋中，通常被攻击的王必须应将。

**将杀(Checkmate)** - 也称“将死”，指攻击对方的王，且被攻击一方没有任何手段可以解除将军。大部分擒王棋中，将杀对方的王即宣告胜利。

**无子可动(Stalemate)** - 在王未被攻击的情况下，没有任何符合规则的行动。在中文中，根据无子可动视为的胜负结果，可分为“逼和”（无子可动作和棋）和“困毙”（无字可动者落败）两种。

**重复行动（循环,Repetition）** - 在对局中某一个局面重复出现，通常以三次为标准。一般这种情况会出现在双方用同一个棋子反复地移动。在不同的棋类中，循环的判别方式各不相同。针对同形反复，不同的版本（比赛规则、软件规则等）有或简洁或细化的不同裁定。

**长将(Perpetual check)** - 一方连续且循环地将军造成的重复行动。不同的棋类中有不同的判法，通常要求实行长将的一方改变行动结束重复，若不变，判法有多种。

**行(Rank)** - 棋盘的横向方向组成的行。

**列(File)** - 棋盘的纵向方向组成的列。

**符号(Notation)** - 在编辑器和棋谱中所用的每个棋子所对应的符号。

**标准代数记法(Standard Algebraic Notation,SAN)** - 在棋类中一种通用的记法。通常为“棋子+目标格”的格式。其余的吃子、打入、升变等，再使用附加符号表示。兵一般省略棋子名。

**回合、步、手** - “回合”指双方各行动一次为一回合，“步”指一个玩家行动一次，“手”指棋子行动一次。在Pychess中，棋类通常为一步一手（Zillions of Games 中收录有一步两手的国际象棋）。在英文中，"Move"的概念根据不同的棋类而改变。国际象棋中的Move意为回合，而将棋中Move指自己行动一次（手）。这也就意味着国际象棋的“一步杀、两步杀、三步杀”分别对应将棋中的“1手诘、3手诘、5手诘”，尽管它们在英文中都被描述为Move。

**打入(Drop)** - 将棋盘外的属于自己的棋子放置在棋盘上的行动。这个术语通常在将棋中出现。带打入的国际象棋变种通常带有"house"的命名，中文则均冠以“狂”字。

**升变区域(Promotion Zone)** - 棋盘上划出的和某个/某些棋子的升变有关的区域。例如国际象棋的对方底线便是兵的升变区域。“升变”通常指棋子的能力改变的事件，每个棋的触发方式不同，如国际象棋只要到达底线则**必须**升变，而将棋则是在进入升变区、在升变区内移动、离开升变区时，均**可选择是否**升变。

# 常见战术

**捉双(Fork)** - 用一个棋子同时攻击对方两个棋子的行为。马因为其独特的走法，非常适合捉双。如果是同时攻击王和另一个棋子，则另一个棋子很大概率会被吃，在中文中被称为“抽”。*（译者注：英文中并未对抽吃这种战术进行细化，它们和捉双统称为Fork。）*

![Fork example](https://github.com/gbtami/pychess-variants/blob/master/static/images/CVariantsGuide/Fork.png)

**牵制(Pin)** - 令一个棋子不能随便移动，否则会暴露后面的高价值棋子被吃。如果后面是王，则被牵制的棋子完全不能离开这条线。

![Pin example](https://github.com/gbtami/pychess-variants/blob/master/static/images/CVariantsGuide/Pin.png)

**串打(Skewer)** - 攻击一个价值较高的棋子迫使其离开，从而吃掉后面价值略低的棋子。与牵制不同的是，牵制是威胁低价值棋子不能离开否则将攻击高价值棋子，而串打是迫使高价值棋子离开从而攻击低价值棋子。

![Skewer example](https://github.com/gbtami/pychess-variants/blob/master/static/images/CVariantsGuide/Skewer.png)

**闪将与闪击(Discovered Attack)** - 将某个棋子移开，露出后面的棋子使其发起攻击。如果后面的棋子攻击的是王，则叫做“闪将”。在中国象棋中，闪将与闪击战术尤为常见。

![Discovered attack example](https://github.com/gbtami/pychess-variants/blob/master/static/images/CVariantsGuide/Discovery.png)

如图，黑马移开实行闪将，同时攻击黑后。因黑方必须应将，丢后已经是必然的了。

**弃子(Sacrifice)** - 为了取得更大的盘面优势，而牺牲部分棋子。

![Sacrifice example](https://github.com/gbtami/pychess-variants/blob/master/static/images/CVariantsGuide/Sacrifice.png)

如图，白选择弃后吃黑马。若黑兵吃后，则白马将直接将杀黑王。
