Gorogoro+ 是将棋的精简版，棋盘大小改为 5x6 。 Gorogoro ごろごろ在日文为状声词，直译为「呼噜呼噜」。

## 规则

由于此游戏基于将棋，请先熟悉将棋的规则。

初始布置如上。

不像将棋，没有飞车、角行，桂马与香车开始时则持有在手上。升变区改为最后两排。
